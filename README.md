# Vibe Project - Android

Android native app built with **Kotlin** and **Material Design 3**.

## Project Structure

```
app/
├── src/main/java/com/vibeproject/
│   ├── MainActivity.kt
│   ├── ui/
│   ├── data/
│   └── domain/
├── src/main/res/
│   ├── layout/
│   ├── values/
│   └── drawable/
└── src/test/
```

## Tech Stack

- **Language**: Kotlin
- **UI**: AndroidX + Material Design 3
- **Architecture**: MVVM
- **Min SDK**: API 24

## Getting Started

1. Install Android Studio
2. Open project and sync Gradle
3. Run on device/emulator
